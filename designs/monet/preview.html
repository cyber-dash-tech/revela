Curatorial note

Large numerals behave like washed layers in the background. The title remains primary; the number should guide the eye, not label it mechanically.

Monet Design System · TOC Study

Usage notes

TOC Panel

  • Tilt number plaques. Use lightly rotated chapter cards instead of utility numerals so the index feels collected, not engineered.
  • Catalogue title stacking. Let the main heading break across two offset lines to create a more composed, art-book-like entrance.
  • Restore TOC semantics. Each row should read as chapter label, title, and page number before anything more decorative is added.
