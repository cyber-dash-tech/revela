Visual schema first

Preserve composition before changing style

  • Reference type. Identify whether the source is vector, photo, UI, page, or motif.
  • Scale and anchor. Keep a bottom strip low, a corner motif small, and a hero visual dominant only when requested.
  • Implementation path. Use SVG for flat motifs; use image/surface rules for photo and UI references.
