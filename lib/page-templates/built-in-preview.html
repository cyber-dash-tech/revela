This key message stays large and readable, while the supporting evidence is separated into traceable slots for source-backed proof.

risks-tradeoffs

Constraint

Templates can over-constrain if they become decorative presets instead of communication jobs.

Mitigation

Keep bounded HTML edit flow after scaffold insertion so agents can improve the page directly.

Tradeoff

More structure improves QA, but only if template contracts stay small and semantic.
